# Row

## Demo

[Demo](https://rike422.github.io/tanmono/?selectedKind=Row) is hosted on GitHub Pages.

## Usage

```jsx
import React from 'react';
import { Row } from 'tanmono';

class ExampleComponent extends React.Component {
  render: () {
    return ( <Row></Row> );
  }
}
```

### Proptypes

| Name | Type | isRequired |
|:----------|:--------:|:--------:|
| reverse | bool | false |
