# Form

## Demo

[Demo](https://rike422.github.io/tanmono/?selectedKind=Form) is hosted on GitHub Pages.

## Usage

```jsx
import React from 'react';
import { Form } from 'tanmono';

class ExampleComponent extends React.Component {
  render: () {
    return ( <Form></Form> );
  }
}
```

### Proptypes

| Name | Type | isRequired |
|:----------|:--------:|:--------:|
| layout | string | false |
